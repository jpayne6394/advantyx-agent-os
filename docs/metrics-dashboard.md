# Agent OS Metrics Dashboard

This dashboard is generated by GitHub Actions and summarizes the current health of the autonomous delivery system.

## What it tracks

- total open pull requests
- merge-ready pull requests
- blocked pull requests
- retries in progress
- drift-detected pull requests
- human escalations
- queue active vs waiting
- priority mix
- failure class mix
- average PR age
- stale PR count

## Refresh model

The dashboard is refreshed by `.github/workflows/metrics-dashboard.yml` on a schedule and by manual dispatch.
